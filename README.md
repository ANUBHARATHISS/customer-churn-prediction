# Customer Churn Prediction

## Project Overview
Built a Machine Learning model to predict which telecom customers 
are likely to churn (leave the service).

## Problem Statement
Customer churn costs companies crores every year. This model helps 
identify at-risk customers before they leave so the business can 
take action.

## Dataset
- Source: Kaggle Telco Customer Churn Dataset
- 7043 customer records, 21 features

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (Random Forest)
- Matplotlib, Seaborn

## Results
- Model Accuracy: 80%+
- Key Finding: Month-to-month contract customers churn 3x more 
  than yearly contract customers
